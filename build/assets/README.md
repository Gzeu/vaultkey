# build/assets/

Place the following files here **before building**. They are gitignored (binary assets).

| File | Format | Used for | Required |
|------|--------|----------|----------|
| `icon.ico` | ICO (256x256 + 128 + 64 + 48 + 32 + 16) | Windows `.exe` icon | Windows only |
| `icon.icns` | ICNS (macOS multi-res) | macOS `.app` + `.dmg` | macOS only |
| `icon.png` | PNG 256x256 | Linux AppImage icon | Linux only |
| `dmg-background.png` | PNG 660x400 | macOS DMG background | Optional |

## Generating icons from a single PNG

```bash
# macOS — from icon-1024.png
mkdir icon.iconset
for S in 16 32 64 128 256 512; do
    sips -z $S $S icon-1024.png --out icon.iconset/icon_${S}x${S}.png
done
iconutil -c icns icon.iconset
rm -rf icon.iconset

# Windows — using ImageMagick
convert icon-1024.png -resize 256x256 icon.ico

# Or online: https://cloudconvert.com/png-to-icns
```

## Notes

- `entitlements.plist` is already tracked in git (XML, safe to commit)
- `version.txt` is auto-generated by `build-windows.ps1`
- `.gitignore` in this folder excludes `*.ico`, `*.icns`, `*.png`, `*.dmg`
